    RxJS 中文文档简介

    关于中文版


    此中文文档会与官方文档 保持同步更新。虽然保持同步的过程或琐碎、或耗费精力，但为了防止过时的内容对读者的学习造成困扰，这份付出还是值得的。

    我们全文采用意译的方式，在确保理解作者意思的前提下用中文重新表述，力求做到“信雅达”，当原文难以直译时更是如此。在必要时，我们会加“译注”来辅助读者阅读。 当然，即便如此，我们理解错误的可能性也还是有的，如有发现，请到我们的 GitHub 上提出 issue 或 Pull Request ，我们不胜感激。

    对于英文词汇，我们尽量采用业内成熟的译法，以利于口头交流。对少部分 RxJS 中的专有名词，仍无法对译文达成广泛共识的，我们会在译文中写成中英双语或保留原文。 这部分做了初步的梳理，请参考 RxJS 词汇表，但是仍可能有遗漏，欢迎前来补充。

    如果你觉得此中文文档对学习 RxJS 有所帮助的话，希望你可以在精神上支持我们，这样我们才能知道我们所做的一切都是有意义的。
