    Notification

    代表可以被 Observable 发出的基于推送体系的事件或者值。对于像 materialize， dematerialize， observeOn 和其他管理通知的操作符来说，Notification 类尤其适用。 除了包装真正发出的值，它还使用元数据进行注解。例如，推送消息的类型是（next，error， or complete）。

    See:

    • materialize
    • dematerialize
    • observeOn
