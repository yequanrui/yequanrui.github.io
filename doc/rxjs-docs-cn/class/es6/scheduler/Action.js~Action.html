    Action

    Extends:

    Subscription → Action

    在调度器（Scheduler）中要执行的任务单元。action 通常是在调度器内部创建，并且 RxJS 用户 不需要关注它的创建和维护。

    class Action<T> extends Subscription {
  new (scheduler: Scheduler, work: (state?: T) => void);
  schedule(state?: T, delay: number = 0): Subscription;
}
