# 前言

Web 应用日益复杂，相关开发技术也百花齐放，这对前端构建工具提出了更高的要求。 Webpack 从众多构建工具中脱颖而出成为目前最流行的构建工具，几乎成为前端开发里的必备工具之一， 因此每位紧跟时代的前端工程师都应该掌握 Webpack。

本书从实践出发，用简单易懂的例子带你快速入门Webpack，再结合实际工作中常用的场景给出最佳实战，通过 前3章的学习足以让你解决工作中的常见问题； 本书还介绍了如何优化构建的速度和输出，并解析了Webpack的工作原理，以及 Plugin和Loader的编写方法，可帮助你进一步学习Webpack。 本书按照入门、配置、实战、优化和原理的路线层层深入，涵盖了Webpack的方方面面。

本文主要基于webpack3版本的特性解析。 由于Webpack从版本1到现在，其核心思想和API没有发生很大的变化，所以文中大部分内容适用于Webpack的任何稳定版本，但部分实例代码只适用于最新版本。
